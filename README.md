# 🎀 Control de Gastos

Aplicación web para registrar y controlar gastos personales. Permite agregar gastos por categoría, mes y monto, ver el total y el promedio, identificar en qué categoría se gasta más y guardar la información automáticamente en el navegador.

## ✨ Características

- ➕ Agregar gastos con categoría, descripción, mes y monto
- 🔤 Mayúscula inicial automática en categoría y descripción
- 🏷️ Sugerencias de categorías comunes (Comida, Transporte, Ocio, etc.)
- 📅 Registro del mes del gasto (mostrado como `mm/aaaa`)
- 💵 Montos con formato de pesos chilenos (ej: `$2.500`)
- 🗑️ Eliminar gastos de la lista
- 💸 Cálculo automático del total gastado
- 🌸 Cálculo del promedio por gasto
- 🏆 Detección de la categoría en la que más se gasta
- 💾 Guardado automático en el navegador (los datos no se pierden al recargar)
- ✅ Validación de datos (no permite categorías vacías ni montos inválidos)
- ⌨️ Permite agregar con la tecla Enter
- 👋 Mensaje de bienvenida que pide tu nombre solo la primera vez
- 📱 Diseño responsive (se adapta a móvil y escritorio)

## 🛠️ Tecnologías

- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## 🚀 Cómo usar la aplicación

1. Descargar o clonar el proyecto.
2. Abrir el archivo `index.html` en el navegador (o usar Live Server en VS Code).
3. La primera vez, un mensaje de bienvenida pide tu nombre y muestra un menú de opciones en la consola.
4. Completar categoría, descripción, mes y monto, y presionar "Agregar ♡" (o Enter).
5. Los gastos aparecen en la tabla, junto con el total, el promedio y la categoría con más gasto.
